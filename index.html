Saat ini, saya aktif mempelajari berbagai bahasa pemrograman seperti [daftar bahasa pemrograman kamu] dan berfokus pada pengembangan keterampilan di bidang AI, sistem digital, dan pemrograman.

Sertifikasi

Saya percaya pembelajaran tidak pernah berhenti. Untuk meningkatkan keterampilan dan kredibilitas, saya telah menyelesaikan berbagai pelatihan dan sertifikasi. Sertifikasi ini membuktikan komitmen saya terhadap pengembangan profesional di dunia teknologi.

Sertifikasi MTCNA MikroTik

Sebagai pemegang sertifikasi MTCNA, saya memiliki kemampuan dalam mengkonfigurasi, mengelola, dan mengoptimalkan jaringan menggunakan perangkat MikroTik. Sertifikasi ini membuktikan keahlian saya di bidang jaringan komputer.

Sertifikasi HTML

Sertifikasi ini menunjukkan pemahaman saya tentang dasar-dasar HTML, termasuk struktur halaman web, elemen-elemen penting seperti tag, atribut, formulir, tabel, dan pengelolaan hyperlink. Saya juga telah mempelajari cara membuat halaman web yang terstruktur dan semantik, memastikan aksesibilitas dan keterbacaan bagi pengguna maupun mesin pencari.

Sertifikasi CSS

Sertifikasi ini menunjukkan kemampuan saya dalam menggunakan CSS untuk mendesain dan mengatur tampilan halaman web. Saya mempelajari cara menggunakan selektor, properti styling, dan layout responsif untuk menciptakan pengalaman pengguna yang menarik dan konsisten di berbagai perangkat.

Sertifikasi JavaScript

Sertifikasi ini menunjukkan kemampuan saya dalam mengembangkan fitur interaktif pada halaman web menggunakan JavaScript. Saya mempelajari konsep dasar pemrograman seperti variabel, fungsi, array, DOM manipulation, dan event handling. Sertifikasi ini juga mencakup logika dasar dan pengenalan ke pengembangan web yang lebih dinamis.
